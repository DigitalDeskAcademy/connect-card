# Archived Documentation

This directory contains documentation that is no longer actively maintained but preserved for historical reference.

## Why Archive Instead of Delete?

- **Git History**: Preserve context for why decisions were made
- **Future Reference**: Historical plans may inform future features
- **Audit Trail**: Track evolution of project documentation

## Active Documentation Location

Current documentation lives in `/docs/` with SSOT (Single Source of Truth) pattern:

- **Core Dashboards**: STATUS.md, ROADMAP.md
- **Feature Visions**: /docs/features/{feature}/vision.md
- **Essential Guides**: /docs/essentials/
- **Technical Decisions**: /docs/technical/architecture-decisions.md
